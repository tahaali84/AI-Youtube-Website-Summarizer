#  AI Youtube Website Summarizer

An AI-powered text summarization application built using LangChain and Groq LLMs that can summarize content from YouTube videos and websites. The application extracts content, processes it using Large Language Models, and generates concise summaries.

##  Features

-  Summarize YouTube videos using transcripts
-  Summarize website articles and webpages
-  Powered by Groq LLM API
-  LangChain-based LLM pipeline
- 📝 Generates concise 300-word summaries
- ⚡ Fast inference using Llama models
- 🎨 Interactive Streamlit UI
